## 1. 产品概述

### 1.1 产品定位

面向能源化工产业链从业者的**基本面价差监控与统计偏离预警工具**。将传统分析师每日 Excel 手工拉数、人工判断的重复劳动，升级为从数据采集到可视化预警的全自动化闭环系统。

### 1.2 解决的痛点

| 痛点 | 传统方式 | 本产品方案 |
|------|----------|------------|
| 多品种价格分散在不同数据源 | 人工逐一查阅 Wind/Reuters/文华 | 统一价格表，一次拉取全部品种 |
| 产业链价差需手动计算 | Excel 公式，容易出错 | 16 组价差自动计算，系数内置 |
| 价差异常全靠经验判断 | 肉眼扫图，无量化标准 | Z-Score 滚动窗口统计 + 三级预警 |
| 离岗无法跟踪行情 | 依赖公司终端 | 公网看板，手机/电脑均可访问 |
| 复盘缺乏系统工具 | 手动翻历史数据 | 多时间窗口走势图 + 热力图 |

### 1.3 目标用户

- **一级用户：** 能化品种研究员/交易员/经纪人（日常监控产业链利润与价差偏离）
- **二级用户：** 风控/合规人员（跟踪关键价差的极端偏离）
- **三级用户：** 面试官/招聘方（展示产品化思维与技术落地能力）

### 1.4 用户场景

#### 场景1：研究员晨间复盘（每日盘前 8:30-9:00）

1. 打开看板 → **行情概览**页 → 扫一眼涨跌幅热力图（近 1 日），了解昨日整体涨跌方向
2. 切换到**价差分析**页 → 预警卡片自动置顶，红色（显著偏离）和黄色（关注）的价差一目了然
3. 点击异常价差 → FLIP 动画跳转到对应走势图，查看 σ 带偏离图中当前值与 ±1σ/±2σ 的位置关系
4. 切换近 1 月 / 近 3 月时间窗口，判断偏离是短期波动还是持续趋势
5. 将关键发现（如 PP-3MA 显著收窄、PDH 利润触及历史低位等）写入晨报或推送交易台

#### 场景2：交易员盘中跟踪（盘中进行中）

1. 某品种盘中异动（如 PP 快速拉涨）→ 打开看板查看 PP-LPG（PDH 利润）当前偏离状态
2. 对比 1 月窗口（短期偏离程度）和 3 月窗口（中期趋势），判断当前利润处于历史什么位置
3. 结合 σ 带图和历史百分位，判断是否处于极端区间，辅助入场/出场决策
4. 参考其他相关价差（如 PP-PE、PP-3MA）交叉验证，避免单指标误判

#### 场景3：周度/月度策略复盘（周末或月末）

1. 打开**数据总览**页 → 按产业链筛选（如只看烯烃链）
2. 查阅价差统计表：逐行比对当前值、均值、标准差、Z-Score、百分位、历史极值
3. 切换到价差历史数据表，选择近 3 月或近 1 年窗口，热力图着色直观展示偏离的全时段分布
4. 区分两类信号：持续偏离（连续多日红/黄）vs 刚触及边界（单日触发），前者更有策略价值
5. 将结论作为下阶段策略输入：哪些产业链利润处于极端区间、哪些价差有回归倾向

#### 场景4：面试官/招聘方评估

1. 打开 GitHub 仓库 → 浏览代码结构、README、PRD 文档
2. 打开在线看板 → 体验行情概览 / 价差分析 / 数据总览三个页面
3. 评估维度：产业链研究框架的完整度（品种覆盖 + 价差逻辑）、产品化落地能力（从数据到部署全链路）、交互设计意识（预警置顶 + FLIP 跳转 + 响应式布局）

## 3. 功能需求

### 2.1 行情概览（P0）

- 12 个能化品种主力合约最新收盘价 + 日涨跌幅卡片展示
- 原油自动桶→吨换算（系数 7.33 桶/吨）
- 多品种主力合约收盘价走势图，支持自选品种、多时间窗口（近 1 月/近 3 月/近 6 月/近 1 年/近 3 年/全部）
- 日度涨跌幅矩阵热力图（红涨绿跌），多时间窗口切换

### 2.2 产业链价差分析（P0，核心模块）

#### 2.2.1 价差指标体系

覆盖 **5 条产业链 × 16 组价差**，系数基于化工工艺物料衡算比例：

| 产业链 | 价差 | 公式 | 经济含义 |
|--------|------|------|----------|
| 🛢️ 原油链 | 沥青-原油 | 沥青 − 原油 | 沥青裂解利润 |
| | 燃油-原油 | 燃油 − 原油 | 燃料油裂解利润 |
| | 原油-LPG | 原油 − LPG | 原料替代相对强弱 |
| 🧪 烯烃链 | PP-LPG | PP − 1.2×LPG | PDH 装置利润 |
| | PP-原油 | PP − 原油 | PP 一体化利润（石脑油路线） |
| | PE-原油 | PE − 原油 | PE 一体化利润 |
| | PP-PE | PP − PE | 聚烯烃相对强弱 |
| 🌸 芳烃链 | PX-原油 | PX − 原油 | PX 一体化利润 |
| | PTA-PX | PTA − 0.655×PX | PTA 加工费 |
| | EB-BZ | EB − 0.8×纯苯 | 苯乙烯加工利润 |
| | EB-原油 | EB − 原油 | 苯乙烯一体化利润 |
| | BZ-原油 | 纯苯 − 原油 | 纯苯一体化利润 |
| 👚 聚酯链 | PTA-EG | PTA − EG | 聚酯原料价差 |
| | EG-原油 | EG − 原油 | 乙二醇一体化利润 |
| 🏭 甲醇链 | PP-3MA | PP − 3.0×甲醇 | MTO/MTP 装置利润 |
| | MA-原油 | 甲醇 − 原油 | 甲醇对原油相对价值 |

#### 2.2.2 统计预警引擎

- **模型：** 基于所选时间窗口的滚动 Z-Score（(当前值 − 均值) ÷ 标准差）
- **三级预警：**
  - ✅ 正常：|Z-Score| < 1.5
  - ⚠️ 关注：1.5 ≤ |Z-Score| < 2.0
  - 🔴 显著偏离：|Z-Score| ≥ 2.0
- **辅助指标：** 均值、标准差、历史最低/最高、历史百分位
- **预警卡片：** 异常卡片置顶高亮，显示当前值 + Z-Score + 百分位，点击自动跳转对应图表（FLIP 动画过渡）
- **σ 带可视化：** 走势图叠加 ±1σ / ±2σ 区间带，均值参考线

### 2.3 数据总览（P1）

- 价差统计表：按产业链筛选，当前值/均值/标准差/Z-Score/百分位/历史极值一览
- 价差说明表：价差公式与经济含义速查
- 价差历史数据表：多时间窗口回溯，热力图着色

### 2.4 自动数据更新（P1）

- macOS launchd 定时任务，每个工作日 15:30（收盘后）自动执行
- 拉取最新日度数据 → 计算价差 → 更新 CSV → 自动推送 GitHub → Streamlit Cloud 同步上线
- 去重逻辑：检测当日数据是否已存在，避免重复拉取
- 周末/节假日自动跳过
- 日志保留 30 天自动清理

### 技术选型理由

| 决策 | 选型 | 理由 |
|------|------|------|
| 数据源 | AKShare（新浪财经） | 免费、覆盖国内期货主力合约、API 稳定 |
| 前端框架 | Streamlit | 纯 Python 即可开发交互式 Web 应用，无需前端技术栈 |
| 可视化 | Plotly Graph Objects | 交互式图表（缩放/悬停/导出），比 Matplotlib 更适合 Web |

| 定时调度 | macOS launchd | 原生支持、休眠唤醒补跑、轻量免安装 |
| 预警模型 | 滚动窗口 Z-Score | 简单可解释、无需训练、适合价差回归特性 |

## 5. 数据规范

### 4.1 品种覆盖

| 品种 | 交易所 | 主力合约代码 | 品类 |
|------|--------|-------------|------|
| 原油 | 上期所 | SC0 | 原油 |
| LPG | 大商所 | PG0 | 液化气 |
| 沥青 | 上期所 | BU0 | 沥青 |
| 燃油 | 上期所 | FU0 | 燃料油 |
| PP | 大商所 | PP0 | 聚丙烯 |
| PE | 大商所 | L0 | 聚乙烯 |
| PTA | 郑商所 | TA0 | 精对苯二甲酸 |
| 甲醇 | 郑商所 | MA0 | 甲醇 |
| PX | 郑商所 | PX0 | 对二甲苯 |
| EG | 大商所 | EG0 | 乙二醇 |
| EB | 大商所 | EB0 | 苯乙烯 |
| 纯苯 | 大商所 | BZ0 | 纯苯 |

### 4.2 数据频率

- **更新频率：** 日度（T日 15:30 更新 T 日数据）
- **历史深度：** 全量保留（从品种上市起）
- **缺失处理：** ffill/bfill 填充节假日断档

### 4.3 价差计算公式校验

- PP-LPG：1 吨 PP 约消耗 1.2 吨 LPG（PDH 工艺物料衡算）
- PTA-PX：1 吨 PTA 约消耗 0.655 吨 PX
- EB-BZ：1 吨 EB 约消耗 0.8 吨纯苯
- PP-3MA：1 吨 PP 约消耗 3 吨甲醇（MTO 工艺）

## 6. 数据存储与流转

### 6.1 数据文件清单

| 文件 | 内容 | 更新方 | 读取方 |
|------|------|--------|--------|
| `data/price_table.csv` | 12 品种日度收盘价，宽表（行=日期，列=品种） | data_fetcher.py | dashboard.py |
| `data/spreads.csv` | 16 组价差日度计算值，宽表（行=日期，列=价差名） | data_fetcher.py | dashboard.py |

### 6.2 数据流向

```
akshare API (新浪财经)
       │
       ▼
data_fetcher.py
  ├── 拉取 12 品种主力合约日度数据
  ├── 清洗: 对齐日期、缺失值填充 (ffill/bfill)
  ├── 原油桶→吨换算 (×7.33)
  ├── 写入/追加: price_table.csv (去重检查)
  ├── 计算 16 组价差
  └── 写入/追加: spreads.csv (去重检查)
       │
       ▼
launchd 定时调度
  ├── 每个工作日 15:30 执行
  ├── 周末/节假日自动跳过
  └── 执行后 git add + commit + push → GitHub
       │
       ▼
Streamlit Cloud
  └── 检测 GitHub 更新 → 自动重新部署
       │
       ▼
dashboard.py (用户访问时加载)
  ├── @st.cache_data(ttl=3600) 缓存加载
  ├── 读取 price_table.csv → Tab 1 行情概览
  ├── 读取 spreads.csv → Tab 2 价差分析 + Tab 3 数据总览
  └── 内存中计算: Z-Score (滚动窗口) / 百分位 / 统计指标
```

### 6.3 读写模式

| 模式 | 说明 |
|------|------|
| 写入 | data_fetcher.py 追加写入（append），单次更新只追加一行 |
| 读取 | dashboard.py 只读加载，st.cache_data 缓存 3600 秒 |
| 并发 | 不存在并发写入（单定时任务，单机执行） |
| 回滚 | 所有数据文件纳入 Git 版本控制，误操作可回滚 |

## 7. 非功能性需求

### 5.1 性能

- 首屏加载时间 ≤ 3 秒（Streamlit cache_data 缓存 3600s）
- 时间窗口切换 ≤ 1 秒（纯前端过滤已加载数据）

### 5.2 可靠性

- 数据拉取异常不中断整体流程（单品种失败不影响其他）
- launchd 休眠唤醒自动补跑（StartCalendarInterval 特性）
- 日志完整记录每次更新的成功/失败状态

### 5.3 可用性

- 响应式布局，PC/手机均可正常浏览
- 预警卡片点击自动跳转对应图表，减少翻找
- 中文界面，面向国内从业者

### 5.4 可维护性

- 单文件架构（dashboard.py ~650 行），易于理解和修改
- 品种/价差配置集中在字典常量，新增品种仅需 3 处改动
- Git 版本控制 + GitHub 备份

## 10. 附录

### 8.1 术语表

| 术语 | 说明 |
|------|------|
| Z-Score | 标准分数，衡量当前值偏离均值的程度（单位：标准差） |
| Crack Spread | 裂解利润，原油加工成产品的利润 |
| PDH | 丙烷脱氢工艺，LPG → 丙烯(PP) |
| MTO/MTP | 甲醇制烯烃/丙烯工艺 |
| FEI/CP/MB | LPG 国际定价基准（远东指数/合同价/蒙特贝尔维尤） |
| Backwardation | 近月升水，近月合约价格高于远月 |
